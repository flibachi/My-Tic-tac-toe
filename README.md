# Крестики-Нолики

Многопользовательское веб-приложение «Крестики-Нолики» с REST API и браузерным интерфейсом: регистрация и вход по логину/паролю, игра с другим пользователем или с компьютером, присоединение к ожидающим партиям и отслеживание состояния игры. Бэкенд построен по слоистой архитектуре (domain / web / datasource) с DI-контейнером, HTTP Basic Auth для защищённых эндпоинтов и хранением пользователей и партий в PostgreSQL.

## Стек

- **Backend:** Python, Flask, SQLAlchemy, Flask-SQLAlchemy
- **База данных:** PostgreSQL (JSONB для игрового поля)
- **Frontend:** HTML, CSS, Vanilla JavaScript (Fetch API)
- **Инструменты:** pytest, Make
